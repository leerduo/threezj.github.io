Java IO 类图

什么是流

  • 一连串有顺序的数据系列可以看成是一个流。
  • Java将文件看成是一个流。
  • 流有一个存取指针，它指向流中可以存取数据的位置。

    流的分类

  1. 输入流和输出流
  2. 字节流和字符流
    • 字节流：按1个字节进行输入输出，直接对文件本身操作
    • 字符流：按2个字节进行输入输出，操作时使用了缓冲区。
  3. 节点流和处理流
    • 节点流：直接连接到数据源
    • 处理流：对一个已存在的流进行连接和封装，拥有更多的方法。并不直接连接到数据源

缓冲技术

  • 缓冲技术是指在内存中开辟一块区域，称之为缓冲区。能提高IO的性能。当缓冲区满的时候一次写入到磁盘中。
  • BufferedInputStream/BufferedOutputStream :用于二进制数据文件
  • BufferReader/BufferWriter：适合于读写文本文件
  • 提供更多的方法
